# TulipFarm.one
Open source ERGO pool.
## Quick start

> `> git clone https://github.com/WinterTFG/TulipFarm.git`<br>
> `> docker-compose up`

Start on TESTNET
> `> docker-compose -f docker-compose.testnet.yml up`
